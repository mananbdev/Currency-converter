Currency Converter is a lightweight web app that lets you instantly convert between 150+ world currencies using live exchange rates. It was built as a web development project to practice real-world API integration, responsive design, and clean UI principles.

**[🚀 View Live Demo](https://mananbdev.github.io/currency-converter/)** | **[📂 View Repository](https://github.com/mananbdev/Currency-converter)**

---

## ⭐ Key Features

* **Live exchange rates:** Rates updated every 24 hours from a reliable API source.
* **150+ currencies:** Covers all major and most minor world currencies with dynamic country flags.
* **Instant conversion:** Results appear instantly without page reloads, with built-in swap functionality.
* **Completely Free:** No signup, no ads, no hidden charges. Just convert.

---

## 🛠️ Built With

* **HTML5** * **CSS3** (Responsive Card UI, Custom Variables, Hover States)

## 🚀 Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/mananbdev/Currency-converter.git](https://github.com/mananbdev/Currency-converter.git)
   ```
2. **Navigate to the directory:**
   ```bash
   cd Currency-converter
   ```
3. **Launch the application:**
   Open the folder in VS Code and start your local server (e.g., using the "Live Server" extension) to open `index.html` in your browser.
