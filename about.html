About the project

Currency Converter is a lightweight web app that lets you instantly convert between 150+ world currencies using live exchange rates. It was built as a web development project to practice real-world API integration, responsive design, and clean UI principles.

Key Features

Live exchange rates

Rates updated every 24 hours from a reliable API source.

150+ currencies

Covers all major and most minor world currencies.

Instant conversion

Results appear as you type, no page reload needed.

Completely Free

No signup, no ads, no hidden charges. Just convert.
